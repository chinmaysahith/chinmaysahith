### 🚀 My Work

#### [llm-powered-multi-agent-system-with-rag](https://github.com/chinmaysahith/llm-powered-multi-agent-system-with-rag)
A locally-hosted, privacy-first AI study assistant. Users upload PDFs or images (OCR-parsed), and the system indexes the content into a vector store. A hybrid intent router — rule-based fast path with an LLM fallback — sends queries to individual agents (Explainer, Question Generator, Doubt Resolver) or chains them into multi-step pipelines (e.g. Summarizer → Question Generator for a full study plan).

**Stack:** Python · Flask · LangChain · Ollama (`phi3:mini`) · FAISS · Tesseract OCR

**What I'd improve next:** centralizing config instead of hardcoded values, and adding a proper `requirements.txt` — I know exactly where the gaps are, which is half the point of building it.
